## 试玩游戏 1：胜利流程（约 18.8 秒）

| 时间       | 画面与事件                                                                                                                                                    |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 0.0–0.5s   | 已处于战斗中：左侧是骨架鱼形初始生物，右侧出现粉白色小鱼群。顶部为关卡标题和“第185关”；右侧有“排行榜、通关记录”。底部是“生物进化”面板，4 个单位均为黑色剪影。 |
| 0.5–3.2s   | 骨架鱼自动朝右侧鱼群攻击/吞噬，敌方鱼群持续从右侧出现。底部单位卡不发生点击反馈，属于自动演示。                                                               |
| 3.3–3.9s   | 骨架鱼触发进化特效，变成“食人鱼”。第 1 张单位卡由剪影替换为真实立绘，并叠加红色“已解锁”。                                                                     |
| 3.9–5.2s   | 食人鱼继续自动攻击右侧鱼群；第 1 张卡片的“经验值”进度条逐步填满为绿色。                                                                                       |
| 5.3–5.8s   | 触发第 2 次进化：“史前蓝鲸”出现；第 2 张卡片由剪影变为真实立绘，并显示“已解锁”。                                                                              |
| 5.8–7.1s   | 史前蓝鲸自动攻击右侧成群敌鱼，画面中有白色攻击轨迹、命中光圈。第 2 张卡经验值逐步增长。                                                                       |
| 7.2–7.8s   | 触发第 3 次进化：“巨型恐鳄”出现；第 3 张卡片解锁。                                                                                                            |
| 7.8–9.1s   | 巨型恐鳄自动攻击右侧敌方鱼群；第 3 张卡的经验值推进。                                                                                                         |
| 9.2–9.7s   | 触发第 4 次进化：“史前巨鳄”出现；第 4 张卡片解锁。顶部新增 Boss 血条，名称为“鱼王”；右侧出现大型刺豚 Boss。                                                   |
| 9.7–14.4s  | Boss 战自动进行：史前巨鳄从左侧攻击刺豚 Boss，持续出现白色攻击轨迹、命中圆环和受击反馈；“鱼王”红色血条持续下降至空。                                          |
| 14.4–15.2s | Boss 消失，战斗区停止；全屏加深暗色遮罩。                                                                                                                     |
| 15.2–16.7s | 出现奖励弹窗：顶部角色插画，中间标题“恭喜获得”，下方展示多格随机道具/装备奖励。                                                                               |
| 16.7–18.8s | 切换到紫色 CTA 页面：顶部“你能吞噬多少鱼？”，中部展示“史前巨鳄”，底部大字“无需下载 点击即玩”。                                                                |

## 试玩游戏 2：失败流程（约 17.9 秒）

| 时间       | 画面与事件                                                                                                                                                       |
| ---------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 0.0–0.5s   | 已处于战斗中：左侧露出大型灰色鳄形敌人，中间是骨架鱼形初始生物，右侧为粉白色鱼群。底部同样是 4 个进化单位，但卡片展示“30级解锁、60级解锁、90级解锁、120级解锁”。 |
| 0.5–3.1s   | 初始骨架鱼自动攻击右侧鱼群；大型灰色鳄形敌人始终停留在左侧，形成后续压迫感。                                                                                     |
| 3.2–3.8s   | 初始生物进化为“食人鱼”；第 1 张卡片切换为真实立绘，右侧按钮由“30级解锁”改为“已解锁”。                                                                            |
| 3.8–5.2s   | 食人鱼自动攻击敌鱼群；左侧灰色鳄形敌人仍在场。                                                                                                                   |
| 5.2–5.8s   | 第 2 个单位“魔纹斑鲸”解锁，卡片显示“已解锁”，单位进入战斗区。                                                                                                    |
| 5.8–7.3s   | 魔纹斑鲸自动攻击右侧鱼群，持续出现攻击轨迹与敌群切换。                                                                                                           |
| 7.4–8.0s   | 第 3 个单位“凶神海鳄”解锁，卡片切换为“已解锁”。                                                                                                                  |
| 8.0–9.3s   | 凶神海鳄自动攻击右侧敌群，画面继续推进。                                                                                                                         |
| 9.4–10.0s  | 第 4 个单位“史前巨鳄”解锁并出现。随后战斗 UI 切换为单体对抗：顶部出现“体力”绿色血条；左下出现“吞噬时间 10”。                                                     |
| 10.0–14.5s | 灰色鳄形敌人从左侧持续攻击史前巨鳄，史前巨鳄不断被击退/缩小；“吞噬时间”从 10 递减至约 6；顶部体力条持续下降直至接近归零。                                        |
| 14.5–15.2s | 史前巨鳄被击败，战斗区停止，页面加深暗色遮罩。                                                                                                                   |
| 15.2–17.9s | 出现失败弹窗：鲨鱼头像、裂纹特效、大标题“挑战失败”、提示“你的体力还需加强”；下方有绿色“重新挑战”和红色“退出挑战”两个按钮。视频结束前未进入 CTA 页面。            |

## 交互观察

* 两段视频中均**未展示手势、点击指引或玩家实时操作**。
* 解锁、进化、攻击、Boss 战、成功/失败结算均表现为固定时间轴自动播放。
* 视频 1 的核心结局是“固定胜利 → 奖励 → CTA”。
* 视频 2 的核心结局是“固定失败 → 重新挑战 / 退出挑战”。
